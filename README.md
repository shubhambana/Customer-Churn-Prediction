# Customer Churn Prediction

## Project Overview

This project predicts whether a customer is likely to churn or not using Machine Learning.

## Dataset

IBM Telco Customer Churn Dataset

## Technologies Used

- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-Learn

## Machine Learning Model

Random Forest Classifier

## Project Workflow

1. Data Cleaning
2. Feature Engineering
3. Data Visualization
4. Train-Test Split
5. Model Training
6. Model Evaluation

## Results

Accuracy: 79.10%
